> ### `mysql` 的聚合函数

| 函数                          | 说明     |
| :---------------------------- | -------- |
| count(`[disetinct|all]` `*`)  | 统计总数 |
| sum(`[disetinct|all]` `<列名>`) | 统计综合 |
| max(`[disetinct|all]` `<列名>`)       | 最大值   |
| main(`[disetinct|all]` `<列名>`)      | 最小值   |
| avg(`[disetinct|all]` `<列名>`)       | 平均值   |
